Monday 4 p.m.–4:30 p.m. in PyData

Tackling Location Based Data Challanges

Eli Safra

Audience level:
Intermediate

Abstract

Geographic data requires special treatment when tackling Data challenges. For example, bucketing latitude/longitude into equal size buckets may be misleading due to the curvature of the earth surface and land usage. At this talk we will discuss what Python libraries can be used for working with Geospatial data and how to use them properly.
