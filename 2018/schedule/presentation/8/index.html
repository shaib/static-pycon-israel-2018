Monday 11:30 a.m.–noon in PyData

"I Abstain" - teaching DL when to shut up

Tsvi Lev

Audience level:
Novice

Abstract

When a single decision by a deep network has significant implications (e.g. Medical, Financial, Safety), accuracy is critical. We show some novel methods (some invented at NEC and some public knowledge) to improve accuracy at the cost of 'not answering' in cases the network model feels uncertain.
